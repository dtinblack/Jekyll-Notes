A function, \(f\), from set \(A\) to set \(B\), is a rule that assigns each element of \(A\) to exactly one element of \(B\), and we write \(f ( a ) = b \) and say that \(f\) maps \(a\) to \(b\). To indicate that \(f\) is a function from \(A\) to \(B\) we write: \(f: A \rightarrow B \).



A function is not complete unless we specify its domain, codoamin, range and rule:
Relation: Set of ordered pairs.
Domain: Set \(A\) on which \(f\) is defined. It represents the set of inputs or first elements and is called the preimage.
Codomain: Set \(B\) in which \(f\) takes its values. It represents the set of possible outputs or second elements.
Range: While the codomain is the set of possible outcomes, the range is the set of actual outcomes, and is called the image.


Some examples:





Definition 3.1 Function. A function \(f(x): A \rightarrow B \) is a mapping of each element of \(A\) to an element of \(B\).

Definition 3.1 Injective Function. An injective function\(f\) is said to be one-to-one, if and only if \(f(a) = f(b)\) suchh that \(a = b \) for all \(a\) and \(b\) in the domain in \(f\).

In other words for a function to be one-to-one every disctinct element in A has a distinct image in B.
