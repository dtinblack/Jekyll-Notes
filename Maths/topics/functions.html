--- title: Functions layout: default ---

A function, \(f\), from set \(A\) to set \(B\), is a rule that assigns each element of \(A\) to exactly one element of \(B\), and we write \(f ( a ) = b \) and say that \(f\) maps \(a\) to \(b\). To indicate that \(f\) is a function from \(A\) to \(B\) we write: \(f: A \rightarrow B \). The set \(A\) is called the domain and set \(B\) is called the codomain.

Definitions:

1. \(f\) is one-to-one or injective if preimages are unique. In this case, \( ( a \neq b ) \rightarrow ( f(a) \neq f(b) ) \)

2. \(f\) is onto or surjective if every \(y \in B \) has a preimage. In this case, the range of the (\f\) is equal to the codomain.

3. \(f\) is bijective if it is surjective and injective ( one-to-one and onto ).

A function is not complete unless we specify its domain, codoamin, range and rule:
Relation: Set of ordered pairs.
Domain: Set \(A\) on which \(f\) is defined. It represents the set of inputs or first elements and is called the preimage.
Codomain: Set \(B\) in which \(f\) takes its values. It represents the set of possible outputs or second elements.
Range: While the codomain is the set of possible outcomes, the range is the set of actual outcomes, and is called the image.
