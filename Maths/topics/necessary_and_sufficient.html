--- title: Necessary and Sufficient Conditions layout: default ---

To help discuss conditional statements lets describe a world of shapes:

A quadrilateral is a plane figure bounded by four straight lines.

A square is a four-sided shape which has all its sides equal and all its angles are right angles.

An oblong is a four sided fiqure which has all of its angles right angles, but not all of its sides equal.

A rhombus is a four-sided shape which has all its sides equal, but its angles are not right angles.

Using the following statements ( a statement can either be true or false )

\(P\) = "the shape is a square" and
\(Q\) = "the shape has four sides of equal length"

Note that \(Q\) should include "and its angles are right angles" but using a truth table it can be shown that nothing is lost with a reduced statement.

Sufficient Conditions - If, then

Case (i): If the shape is a square, then it has four sides of equal length..

From the description of a square the statement is true.

Case (ii): If the shape is a square, then it doesn't have four sides of equal length.

From the description of a square the statement is false because must have four equal sides.

Case (iii): If the shape is not a square, then it has four sides of equal length..

Although the shape may not be square it could still have four equal sides e.g. a rhombus, therefore the statement is true.

Case (iv): If the shape is not a square, thenit does not have four sides of equal length.

From the description of a square the statment is true.


The condition of P being true is enough ( i.e. sufficient ) to make Q true; hence "\(P\) is a sufficient condition for \(Q\)." In the world of shapes then being a square is a sufficent condition for it to have four sides.

Necessary Conditions - only if

Using the following Truth Table:


\( % line space \)

\( \begin{array}{|c c|c| c } % |c c|c| means that there are three columns in the table and % a vertical bar ’|’ will be printed on the left and right borders, % and between the second and the third columns. % The letter ’c’ means the value will be centered within the column, % letter ’l’, left-aligned, and ’r’, right-aligned. P & Q & Q \implies P\\ % Use & to separate the columns \hline % Put a horizontal line between the table header and the rest. T & T & T & ( i )\\ T & F & T & ( ii )\\ F & T & F & ( iii )\\ F & F & T & ( iv )\\ \end{array} \)


Case (i): If the shape has four sides, then the shape is a square.

From the definiion of a square the statement is True.

Case (ii): If the shape has four sides, then the shape isn't a square.

Although the shape has four sides it doesn't have to be a square, for example a rhombus, therefore the statement is true.

Case (iii): If the shape doesn't have four sides, then the shape has four sides of equal length.

From the definition of a square the statement is false.

Case (iv): If the shape does not have four sides of equal length, then the shape is not a square

From the defintion of a square then the statement is true.


Note that the statement \(P \implies Q\) being true means that it is impossible that \(P\) is true but \(Q\) is false, so in order for \(P\) to be true it is necessary that \(Q\) is true; hence "\(Q\) is a necessry condition for \(P\)." And this means that \(P\) can only be true if \(Q\) is true i.e., "\(P\) only if \(Q\)" for the example, to have four sides is a necessary condition for the shape to be a square.

Necessary and Sufficient Conditions - If, and only if


Using the following Truth Table:

Case (i): If the shape is a square, then the shape has four sides of equal length AND If the shape has four sides, then the shape is a square .

The statement is True.

Case (ii): If the shape is a square, then the shape doesn't have four sides of equal length.ANDIf the shape has four sides, then the shape is a square

The statement is False.

Case (iii): If the shape is not a square, then the shape has four sides of equal lengthANDIf the shape has four sides, then the shape is a square.

The statement is True.

Case (iv): If the shape is not a square, then "the shape does not have four sides of equal length ANDIf the shape has four sides, then the shape is a square.

The statement is False


If P, then Q being true, the converse statment "If Q, then P" is also true.
