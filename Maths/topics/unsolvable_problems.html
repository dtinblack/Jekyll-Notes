In Alan Turing's article: Solvable and Unsolvable Problems he developes an argument to show that that certain types of problem are unsolvable.

The Theorem to be proven is: There is no systematic proceedure by which one can tell whether a puzzle is solvable.

Turing will uses proof by Reducto Absurbum which attempts to establish a claim showing that the opposite scenario would lead to an absurity or contradiction.

(i) Let \(P(R,S)\) be the puzzle whose rules are described by the row of sympbols \(R\) and whose starting position is described by \(S\).

(ii) There is no reason why we cann't let the rules be the starting position i.e \(P(R,R)\).

(iii) The moves ( or rules ) are unambiquous.

(iv) A puzzle 'comes out' if it reaches th epositiobn \(B\) or \(W\).

The rules of the puzzle can be classified into the following two classes:

Class I: is to consist of sets rules \(R\), which represent puzzles whith unambiquous moves and such that \(P(R,R)\) comes out as \(W\).

Class II: contains all of the other cases.

Proof:

Assume that we have a proceedure that is able to distinquish rules of Class I from Class II. The systematic proceddure fr distinquishing between the two classes can be put in the form of a subsitition puzzle, \(K\) therefore \(P(K,R)\).

Since the proceedure always gives an answer then the puzzle \(P(K,R)\) always comes out. Assume that the puzzle \(K\) gives a result \(B\) for rules of Class I, and \(W\) for rules of class II. In sumary the puzzle with rules \(K\) has the following properties:

(i) \(K\) has unambiquous moves.

(ii) \(P(K,R)\) always comes out whatever \(R\).

(iii) If \(R\) is in class I, then \(P(K,R)\) hs end result \(B\).

(iv) If \(R\) is in class II, then \(P(K,R)\) hs end result \(W\).

If we ask which class \(K\) belongs to we find that the properties are inconsistent with the definitions of the two classes.
